About "Volty"

Hi, i'm Vlasis, a web developer & programmer. Volty is one of the projects i'm working on these days. It's a video player i developed, using the HTML5 video element, CSS3 and Javascript. Currently, it is fully supported on Google Chrome and Mozilla Firefox.

Version: 1.0
Date released: January 05, 2016
Languages used: HTML5, CSS3 & Javascript.
Fully tested & supported on: Google Chrome, Mozilla Firefox.
Bugs / known issues: Not working properly on Opera browser. Probably i'll fix this bug in one of the upcoming versions, in the future.
